1. How to commit code
$ git status : show the change that you have made
$ git add -A : add all change to the git cache
$ git commit -m "<some message>": commit this stage of changes with the commit message
$ git push origin master: push the commits to heroku server, and it will be updated in internet website


2. How to update code
$ git pull origin master: pull the code from heroku server which QQ has made.
